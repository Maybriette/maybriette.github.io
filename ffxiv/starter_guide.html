Getting Started in FFXIV

A quick-start guide to join us in game

1
Sign up for the free trial

Create a Square Enix account and register for the free trial. When asked for your platform, choose Mac.

square-enix.com → Free Trial Registration ↗
2
Download and install
Final Fantasy XIV Mac Download ↗
3
Create a character on our server

We are on Midgardsormr (Aether Data Center).

4
Play the free trial

The free trial has no time limit — play until you're ready to subscribe.

5
Buy the full game

Purchase the Starter Edition to receive a license code, then log into Mogstation and go to Transfer to Full Game → Registration Code Entry to activate it.

Square Enix Store → Starter Edition ↗ mogstation.com → Activate license ↗
6
Enter the Recruit a Friend code

Enter this after activating the full game. It unlocks bonus items for you!

F7CNMWCU valid until 11/3/26
SE guide: how to enter Recruit a Friend code ↗

If the code has expired by the time you get here, just let us know!
